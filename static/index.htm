3D Tech 203

lecturer: Rouan van der Ende - rouan@virtualwindow.co.za

Welcome. In this module we'll explore a new medium. 3D delivered to any device. With applications such as games, entertainment and interactive information.

SublimeText text editor
json exporters for blender, max and maya

We're going to be working on an online music experience for yogazelle. See the look reference below. You are free to stay as close to this style as you like, we are focussing on learning the techniques in a short time.

Lesson 1 - Your First Cube In this lesson we'll learn how to set up our scene, camera and render out our first object inside the browser. You'll learn how to program the position, scale and rotation of objects as well as the camera.
Lesson 2 - Interaction We'll cover some basics of javascript. For loops, while loops, if and else conditionals. We see how we can build controls for our scene using the keyboard and mouse. We introduce you to functions and how to make our own rgb color generator.
Lesson 3 - Particles We create our first particle system and learn how to use sprites.
Lesson 4 - 3D import We install the threejs json exporter addon for blender, and export our first object ready for web.
Lesson 5 - Sound We learn how to make things move to sound. Import mp3 and connect your scene up to visualise the amplitude.
Coming soon Lesson 6 - Shaders We introduce vertex and fragment shaders. How to offload processing to the graphics card and use the CPU for other things.
Coming soon Lesson 7 - Morphing and bones We continue with importing assets, with animation this time. We handle the vertex and bone data directly.

Three.js

We will be focussing on three.js to render visuals and 3D in the browser. It is code based and could be intimidating to those who have not programmed before. You can access the threejs documentation locally.
